# Beveiliging

## Kwetsbaarheden melden

Meld beveiligingsproblemen **niet** via een publieke GitHub-issue.

- **GitHub**: gebruik het tabblad **Security** van deze repository en kies **Report a vulnerability** (private security advisory).
- **Anders**: neem rechtstreeks contact op met de maintainers van dit project.

Vermeld bij voorkeur:

- korte beschrijving en impact
- stappen om het probleem te reproduceren
- eventuele suggestie voor een oplossing

Geen proof-of-concept of exploit in het openbaar plaatsen vóór er een afgesproken fix of embargo is.

## Scope

Deze richtlijn geldt voor deze applicatie (backend, frontend, deployment-config) en de manier waarop die in productie wordt gebruikt. Voor kwetsbaarheden in upstream dependencies: meld die bij voorkeur bij het betreffende project; je kunt ons informeren als ze Lunchkraam direct raken.
